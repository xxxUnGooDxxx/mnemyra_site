# Mnemyra — лендинг-каркас

Статическая одностраничная версия без сборщика и зависимостей. Для просмотра откройте `index.html` в браузере.

## Файлы

- `index.html` — структура и тексты;
- `styles.css` — адаптивный дизайн и анимации;
- `script.js` — мобильное меню и появление секций;
- `logo.svg` — фирменный логотип из Android-проекта.
- `og-image.png` — карточка для соцсетей и мессенджеров;
- `site.webmanifest` — описание сайта для браузеров и Android;
- `robots.txt` — правила обхода поисковыми роботами;
- `SEO_DEPLOY.md` — действия после выбора публичного домена.

Кнопка скачивания пока является заглушкой: перед публикацией ей потребуется ссылка на магазин или APK.
